# ぱくぱくタイマー 要件メモ

## アプリの目的

子どもが楽しみながらご飯を食べられる、iPad横向き用の食事タイマーアプリ。
タイマーが進むたびに中央のキャラクター「パクパクくん」の口がパクパク動き、食べ進めている感じが視覚的にわかる。

## 画面

- iPad横向きで使う。縦向きでも横スクロールなしで操作できる。
- iPadのホーム画面に追加したとき、アプリ名とテーマカラーが反映される。
- GitHub PagesなどHTTPSで公開した場合、初回読み込み後は主要ファイルをキャッシュし、Macを閉じていてもiPadから再表示できる。
- デザインは生成したデザイン案の雰囲気を維持する。
- フォルダ内の参照画像 `assets/references/pakupaku-timer-design-reference.png` をデザインの基準にする。
- アプリ画面は、参照画像の見た目・雰囲気・配置にできるだけ忠実に作る。
- 画面上部にタイトル「ぱくぱくタイマー」を表示する。
- タイトルの下に、全体の残り時間を表示する。
- 中央に大きな丸いタイマーを表示する。
- 大きな丸は3つの食べ物エリアに分ける。
- 画面全体の背景に、小さな食べ物柄を均等に表示する。

## 食べ物エリアの配置

- 左: ヨーグルト
- 真ん中: 卵焼き
- 右: ご飯

## タイマーの動き

- 各エリアは、その食べ物に割り当てた時間に合わせてだんだん減っていく。
- ご飯、卵焼き、ヨーグルトの順に食べ進める。
- ご飯、卵焼きが終わるたびに自動では次へ進まず、区切り確認待ちでタイマーを止める。
- 区切り確認待ちでは「ごはん おしまい？」「つぎは 卵焼き」のように、終わった食べ物と次の食べ物を表示する。
- 区切り確認待ちで「つぎへ」を押すと、次の食べ物からタイマーを再開する。
- 区切り確認待ちで「もう1分」を押すと、終わった食べ物の現在のタイマー時間だけを1分延長して、同じ食べ物を続ける。
- 「もう1分」の延長分は設定値には保存しない。
- ヨーグルトが終わったときは区切り確認の「つぎへ」は出さず、完了表示へ進む。
- 全体の時間表示は、タイマーの進行に合わせて残り時間として減っていく。
- 各食べ物ごとの残り時間も表示する。
- すべて終わったら、食事タイマー完了としてわかりやすく表示する。
- 大きな遅延やバックグラウンド復帰で複数区間をまたいだ場合も、一度に飛ばさず最初に到達した区切りで止める。

## パクパクくん

- 大きな丸の中心に「パクパクくん」を表示する。
- タイマーが動いている間、口を開いたり閉じたりしてパクパクする。
- 口の動きは、食べている楽しさが伝わるようにする。
- 区切り確認待ち中は通常のパクパク動作を止める。
- 完了時は通常のパクパク状態を止め、正面向きのニコニコ顔で食べ終わったことが伝わる表情にする。

## 音

- ご飯、卵焼き、ヨーグルトがそれぞれ終わるたびに「ぱくっ」と音が鳴る。
- 音は短く、楽しい印象にする。
- 子どもが驚きすぎない音量・音色にする。

## 設定

- 同じ画面内の設定パネルで、それぞれの時間を1〜60分のプルダウンから変更できるようにする。
- 設定できる項目:
  - ご飯の時間
  - 卵焼きの時間
  - ヨーグルトの時間
- 設定パネルを開いたときは、現在タイマーに適用中の時間をプルダウンに表示する。
- 設定パネルを開いている間にプルダウンを変更しても、タイマー表示はまだ変更しない。
- 設定ボタンの再タップ、または設定パネル外側のタップでパネルを閉じたとき、選択中の時間をタイマーへ反映する。
- 閉じたときに設定値が変わっていた場合だけ、全体の時間表示と各食べ物の残り時間を更新する。
- 閉じたときに設定値が変わっていた場合だけ、テストモードは解除される。
- 設定値が変わった場合は、区切り確認待ちとその場限りの「もう1分」延長を解除する。

## 現在の確定仕様

- 食べ物エリアの配置:
  - 左: ヨーグルト
  - 真ん中: 卵焼き
  - 右: ご飯
- 食べる順番: ご飯 -> 卵焼き -> ヨーグルト
- 初期時間:
  - ヨーグルト: 4分
  - 卵焼き: 5分
  - ご飯: 5分
- 全体時間: 3つの時間の合計をもとにした残り時間
- 操作ボタン:
  - スタート: タイマーを開始する。
  - とめる: タイマーを一時停止する。
  - もう一回: 完了後に最初から開始する。
  - リセット: タイマーを初期状態に戻す。
  - テスト: 各食べ物を5秒ずつにして動作確認できる。
- 区切り確認:
  - ご飯と卵焼きの終了時に、タイマーを停止して確認パネルを表示する。
  - つぎへ: 次の食べ物から再開する。
  - もう1分: 終わった食べ物を現在のタイマー状態だけ1分延長して再開する。
  - リセット、テスト、設定変更時は確認待ちを解除する。
- 完了表示:
  - 画面中央に「ごちそうさま!」「ぜんぶ食べたよ」を表示する。
  - パクパクくんは完了表情になる。
- 終了音・区切り音: 食べ物の切り替わりと完了時に、アプリ内で生成した短い「ぱくっ」音を鳴らす。
- 音は画面内の音ボタンでオン・オフできる。
- iPad Safariで音が鳴るように、スタートや音ボタンのタップ時に音声再生の準備をする。
- オフライン再表示用に、HTML、CSS、JavaScript、マニフェスト、アイコンをService Workerで保存する。
- デザイン方針: フォルダ内の参照画像をベースに、その通りの見た目を目指す。
- 画面全体の背景: おにぎり、ごはん、卵焼き、ヨーグルト、果物、野菜などの小さな食べ物柄を淡く75個程度、均等に表示する。

- UI、タイマー挙動、音、設定、テストモード、完了表示を変更した場合は、同じ修正でこの仕様書も更新する。
- 仕様未確定の仮実装は、確定仕様にせず「今後決めること」に残す。
- 仕様と実装が食い違った場合は、修正PRまたは作業単位内で必ずどちらかに揃える。
