# ぱくぱくタイマー 要件メモ

## アプリの目的

子どもが楽しみながらご飯を食べられる、iPad横向き用の食事タイマーアプリ。
タイマーが進むたびに中央のキャラクター「パクパクくん」の口がパクパク動き、食べ進めている感じが視覚的にわかる。

## 画面

- iPad横向きで使う。
- デザインは生成したデザイン案の雰囲気を維持する。
- フォルダ内の参照画像 `assets/references/pakupaku-timer-design-reference.png` をデザインの基準にする。
- アプリ画面は、参照画像の見た目・雰囲気・配置にできるだけ忠実に作る。
- 画面上部にタイトル「ぱくぱくタイマー」を表示する。
- タイトルの下に、全体の残り時間を表示する。
- 中央に大きな丸いタイマーを表示する。
- 大きな丸は3つの食べ物エリアに分ける。

## 食べ物エリアの配置

- 左: ヨーグルト
- 真ん中: 卵焼き
- 右: ご飯

## タイマーの動き

- 各エリアは、その食べ物に割り当てた時間に合わせてだんだん減っていく。
- ご飯、卵焼き、ヨーグルトの順に食べ進める。
- ご飯、卵焼き、ヨーグルトのそれぞれが終わるたびに、次の食べ物へ進む。
- 全体の時間表示は、タイマーの進行に合わせて残り時間として減っていく。
- 各食べ物ごとの残り時間も表示する。
- すべて終わったら、食事タイマー完了としてわかりやすく表示する。

## パクパクくん

- 大きな丸の中心に「パクパクくん」を表示する。
- タイマーが動いている間、口を開いたり閉じたりしてパクパクする。
- 口の動きは、食べている楽しさが伝わるようにする。
- 完了時は通常のパクパク状態を止め、正面向きのニコニコ顔で食べ終わったことが伝わる表情にする。

## 音

- ご飯、卵焼き、ヨーグルトがそれぞれ終わるたびに「ぱくっ」と音が鳴る。
- 音は短く、楽しい印象にする。
- 子どもが驚きすぎない音量・音色にする。

## 設定

- 同じ画面内の設定パネルで、それぞれの時間を分単位で変更できるようにする。
- 設定できる項目:
  - ご飯の時間
  - 卵焼きの時間
  - ヨーグルトの時間
- 設定を変えると、全体の時間表示も自動で更新される。
- 設定値を変更したら、テストモードは解除される。

## 現在の確定仕様

- 食べ物エリアの配置:
  - 左: ヨーグルト
  - 真ん中: 卵焼き
  - 右: ご飯
- 食べる順番: ご飯 -> 卵焼き -> ヨーグルト
- 初期時間:
  - ヨーグルト: 4分
  - 卵焼き: 7分
  - ご飯: 9分
- 全体時間: 3つの時間の合計をもとにした残り時間
- 操作ボタン:
  - スタート: タイマーを開始する。
  - とめる: タイマーを一時停止する。
  - もう一回: 完了後に最初から開始する。
  - リセット: タイマーを初期状態に戻す。
  - テスト: 各食べ物を5秒ずつにして動作確認できる。
- 完了表示:
  - 画面中央に「ごちそうさま!」「ぜんぶ食べたよ」を表示する。
  - パクパクくんは完了表情になる。
- 終了音・区切り音: 食べ物の切り替わりと完了時に、アプリ内で生成した短い「ぱくっ」音を鳴らす。
- 音は画面内の音ボタンでオン・オフできる。
- デザイン方針: フォルダ内の参照画像をベースに、その通りの見た目を目指す。
